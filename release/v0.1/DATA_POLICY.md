# Polityka danych i źródeł

- W Git przechowujemy metadane, ekstrakty dozwolone licencją, obliczenia i sumy kontrolne.
- Pełne kopie materiałów zewnętrznych trafiają do `research/cache/`, które jest ignorowane przez Git, chyba że licencja jawnie pozwala na redystrybucję.
- Każdy dataset ma właściciela, datę pozyskania, definicje pól, jednostki, zakres i ograniczenia.
- Dane syntetyczne są zawsze oznaczone `synthetic: true`.
- Nie pobieramy ani nie gromadzimy danych osobowych na zapas.
- Publiczny eksport zawiera wyłącznie materiały z potwierdzonym prawem publikacji.
- Wszystkie artefakty wskazują datę stanu wiedzy.
