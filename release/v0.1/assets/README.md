# Materiały wizualne

`assets/generated/` zawiera gotowe elementy używane przez pipeline. `assets/prompts/` przechowuje pełne prompty, a `assets/metadata/` — pochodzenie, rozmiar, sumę kontrolną, zakres dozwolonego użycia i ograniczenia.

- `delivery-cycle.svg`, jego angielski odpowiednik `delivery-cycle-en.svg` i `roadmap-horizons.svg` są deterministycznymi infografikami wektorowymi zbudowanymi w repozytorium.
- `cover-gpt-image.png` jest syntetyczną ilustracją koncepcyjną wygenerowaną przez platformowe narzędzie OpenAI imagegen. Narzędzie nie ujawniło dokładnej wersji modelu, dlatego metadane nie przypisują mu niepotwierdzonej etykiety „GPT Image 2”.

Każdy materiał wizualny wymaga ludzkiej akceptacji przed publikacją. Obraz koncepcyjny nie jest dowodem, mapą rzeczywistych lokalizacji ani zobowiązaniem infrastrukturalnym.
